# Image to GIF Converter

A simple and free online tool to convert images into GIF format.

All processing is done locally in your browser.
No image upload. No tracking. Privacy-friendly.

## Features

- Convert images (PNG, JPG, WebP, etc.) to GIF
- High Quality / Lightweight mode
- 100% client-side processing
- Free to use
- Works directly in your browser

## Demo

- English: https://ktsuchimoto524-collab.github.io/image_GIF_allinone/en/
- Japanese: https://ktsuchimoto524-collab.github.io/image_GIF_allinone/ja/

## How to Use

1. Open the page
2. Select an image file
3. Choose quality mode
4. Click "Create GIF"
5. Download the generated GIF
